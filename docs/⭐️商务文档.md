# 项目介绍

## 项目简介

**做一个知识产权与合规服务平台，实现服务在线成交与案件流程可视化交付**

## 项目背景

客户是做合规服务的，需要开发一个系统，把商标 / 专利这些案件流程，从原来靠微信、Excel记录的方式，变成一个可以实时看到进度、可追溯、客户也能参与查看的平台
