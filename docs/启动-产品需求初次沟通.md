# 产品背景

## 产品定位

本产品的核心定位是面向全球一站式合规服务平台，实现服务在线成交与案件流程可视化交付。帮助解决出海全链路合规痛点，降低成本、提升效率、防控风险，

## 目标人群

本产品面向的目标人群是有跨境合规服务需求的用户。

## 目标市场

中国。
